# Writing-an-interpreter-in-Go-cs

[「Go言語でつくるインタプリタ」](https://www.amazon.co.jp/dp/4873118220)をC#で写経してみた結果。

## Description

そのままGoで写経するのもなんなので最近のC#の機能の学習もかねてC#でやってみた。

## Requirement

- Visual Studio 2019
  - 無駄にswitch式を使っているのでC# 8.0
- [xunit.net](https://xunit.net/)
